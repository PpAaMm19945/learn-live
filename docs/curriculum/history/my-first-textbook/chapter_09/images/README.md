# Image Assets

This directory contains all the images used in this chapter.

## Naming Convention

To keep things organized, please use the following naming convention for images:

`chapter-XX-figure-YY-description.jpg`

For example: `chapter-01-figure-01-table-of-nations.jpg`

## Image Specifications

To ensure a consistent and high-quality visual experience in both the PWA reader and the PDF export, please adhere to the following guidelines:

*   **Full-width images:** These should be at least 1200px wide.
*   **Inline images:** These should be at least 600px wide.
*   **Resolution:** Aim for a resolution of 72 DPI for web images.
*   **File format:** Use `.jpg` for photographs and `.png` for graphics or images with transparency.
